# True Colors Personality Test

A modern, responsive web application that allows users to take the "True Colors" personality test. Built with React, Vite, and Tailwind CSS, this project provides an engaging user experience from start to finish.

## Features

- **Interactive Questionnaire**: A 20-question test designed to determine the user's personality type.
- **Personalized Results**: A detailed results screen that reveals the user's dominant color, provides a full score breakdown, and lists personal strengths and potential challenges.
- **Sleek, Modern UI**: A "fancy" and polished user interface built with Tailwind CSS, focusing on a great user experience.
- **Fully Responsive**: The application is designed to work beautifully on all devices, from mobile phones to desktop computers.
- **Accessible Design**: Built with accessibility in mind, featuring high-contrast colors and clear focus states.

## Tech Stack

- **Frontend**: [React](https://reactjs.org/) (with Hooks)
- **Build Tool**: [Vite](https://vitejs.dev/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (version 14 or later)
- npm or pnpm

### Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/ranjithprabhuk/personality-test-true-colors.git
   ```
2. Navigate to the project directory:
   ```sh
   cd personality-test-true-colors
   ```
3. Install NPM packages:
   ```sh
   npm install
   ```

### Running the Application

To start the development server, run the following command:

```sh
npm run dev
```

This will start the application, and you can view it in your browser at `http://localhost:5173` (or the port specified in your terminal).

## Deployment
